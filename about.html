About Me

Selamat datang di halaman tentang saya! Saya adalah seorang web developer yang berdedikasi dalam membangun website modern, interaktif, dan user-friendly.

Saya memiliki latar belakang pendidikan di bidang Informatika dan telah berpengalaman dalam mengerjakan berbagai project web development baik skala kecil maupun besar, menggunakan HTML, CSS, JavaScript dan framework modern seperti Bootstrap dan React.

Selain coding, saya juga menyukai desain UI/UX dan sangat peduli dengan tampilan yang responsif serta estetika layout yang nyaman di semua perangkat.

Tujuan saya adalah terus berkembang dan membantu bisnis ataupun personal project memiliki website yang fungsional sekaligus menarik secara visual. Jangan ragu untuk mengeksplorasi halaman lain dan hubungi saya melalui halaman kontak.
